The Platform Overview is the front door to the ETIS Engineering Platform. It explains what the Starter Kit is, how to use it, how the Engineering Stages fit together, and how the repository is organized before you begin ES-100.

## How the pieces fit together

<div class="etis-flow-panel">
<pre><code>ETIS Framework
      ↓
ETIS Starter Kit Repository
      ↓
Engineering Platform + Templates + Examples
      ↓
docs/project-workspace/
      ↓
Trustworthy Intelligent System
      ↓
Post-Release Learning and Stewardship</code></pre>
</div>


The Framework explains the discipline. The Starter Kit helps you practice it. The `docs/project-workspace/` directory is where cloned-project engineering evidence is created and maintained.

When you are ready to work on a real project, open the [Project Workspace](../project-workspace/README.md). It is intentionally linked from here because the Platform Overview is the place where readers move from learning the platform to using it.

<div class="etis-ui-callout insight">
  <strong>Engineering Insight</strong>
  <p>The Starter Kit is not a document collection. It is a repository-centered engineering workbench for creating durable evidence inside the project workspace.</p>
</div>
