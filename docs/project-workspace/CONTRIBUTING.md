# Contributing

This repository uses ETIS repository-centered engineering.

## Pull request expectations

- Explain what changed.
- Explain why it changed.
- Update affected lifecycle evidence.
- Link related requirements, decisions, risks, tests, release notes, or governance records.
- Record material AI assistance where appropriate.
- Identify remaining risks or follow-up actions.

## Update before publishing

Add project-specific review rules, branch policy, reviewer expectations, and merge criteria.
