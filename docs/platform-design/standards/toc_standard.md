# Table of Contents Standard

The right-side Table of Contents is part of the engineering workspace.

It should function as a live outline of the page.

## Heading rules

- H1: page title only.
- H2: major page sections.
- H3: subtopics only when useful.
- Avoid H4+ unless necessary.

## TOC quality

A good TOC should let an engineer understand the page structure without reading the whole page.
