# Navigation Standard

Navigation is part of the engineering experience.

Every page should make movement obvious.

## Required navigation

Engineering pages use:

- previous page;
- current page;
- next page;
- final continue action.

## Link rules

Use MkDocs generated-page links when linking inside rendered pages.

For pages inside an ES directory:

```text
../navigation/
../readiness_gate/
../../ES-101/
```

Avoid raw `.md` links inside HTML components.
