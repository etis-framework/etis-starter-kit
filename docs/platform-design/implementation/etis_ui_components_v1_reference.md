## Gate Checklist

<div class="etis-gate-checklist">
  <div class="etis-gate-checklist-title">Gate Criteria</div>
  <ul>
    <li>The purpose of ES-100 is understood.</li>
    <li>The Engineering Stage model is understood.</li>
    <li>The user understands that templates are not evidence until completed and reviewed.</li>
