# Page Archetypes

The platform uses a small set of repeatable page archetypes.

## Narrative Page

Teaches concepts and philosophy.

## Reference Page

Defines terminology, maps, structures, or standards.

## Operational Page

Guides action, review, evidence creation, or readiness.

## Manifest Page

Defines a stage contract.

## Readiness Gate Page

Evaluates whether the engineer or team is ready to advance.
