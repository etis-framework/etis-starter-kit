# ETIS Platform Design System

The ETIS Platform Design System defines the product experience, visual language, page architecture, navigation model, component system, migration strategy, and governance model for ETIS product properties.

This design system exists so ETIS products do not become disconnected collections of pages. It provides the engineering discipline needed to make every ETIS page recognizable, navigable, purposeful, and consistent.

## Purpose

The design system answers four questions for every ETIS product page:

1. Where am I?
2. Why am I here?
3. What should I do?
4. What comes next?

If a page does not answer those questions, it is not finished.

## Scope

The design system applies to:

- ETIS Framework website
- ETIS Engineering Platform Starter Kit
- ETIS Instructor Resources
- ETIS Professional Toolkit
- ETIS Review Board Playbook
- Future ETIS ecosystem products
