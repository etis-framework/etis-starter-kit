# Architecture Decisions

This file records major platform architecture decisions.

## ADR-001 — ES-100 is the reference implementation

ES-100 defines the first production implementation of the Engineering Stage experience.

## ADR-002 — Right-side TOC is retained

The right-side TOC functions as a live engineering outline.

## ADR-003 — New UI patterns require platform approval

Future stages may not introduce new UI patterns unless the platform architecture is updated first.

## ADR-004 — Platform design remains internal

`docs/platform-design/` is repository-internal architecture material and is not added to public MkDocs navigation at this phase.
