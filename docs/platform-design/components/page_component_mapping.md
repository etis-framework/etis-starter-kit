# Page-to-Component Mapping

This document maps ETIS page types to required and optional components.

## Landing Page

Required:

- Product Hero
- Primary Navigation Cards
- Audience Path Cards
- Relationship Flow
- Final CTA

Optional:

- Metrics / Stats
- Principle Chips
- Repository Structure Panel

## Section Landing Page

Required:

- Section Hero
- Navigation Cards
- Recommended Path
- Next Activity Panel

Optional:

- Relationship Flow
- Concept Cards
- Frequently Used Areas

## Engineering Stage Overview

Required:

- Stage Banner
- Metadata Strip
- Lifecycle Position
- Stage Outputs
- Stage Navigation
- Next Activity Panel

Optional:

- Evidence Panel
- Related Templates
- Related Examples

## Engineering Activity Page

Required:

- Stage Banner or Page Header
- Page Purpose Block
- Activity Guidance
- Evidence Reminder
- Next Activity Panel

Optional:

- Engineering Insight
- AI Responsibility Callout
- Metadata Strip
- Example Link Panel

## Readiness Gate Page

Required:

- Readiness Gate Header
- Completion Standard
- Gate Checklist
- Evidence Panel
- Gate Decision Panel
- Next Activity Panel

Optional:

- Remediation Panel
- Governance Callout
- Stage Closure Panel

## Template Page

Required:

- Template Purpose
- When to Use
- Inputs
- Outputs
- Blank Template
- Completion Guidance

Optional:

- Example Link Panel
- Evidence Reminder
- Common Mistakes Panel

## Example Page

Required:

- Example Context
- Artifact Purpose
- What to Notice
- Completed Example
- Adaptation Guidance

Optional:

- Traceability Panel
- Related Template Link
- Review Notes

## Reference Page

Required:

- Scope
- Reference Content
- Cross-links if relevant

Optional:

- Glossary Link Panel
- Related Resources Panel

## Checklist Page

Required:

- Checklist Purpose
- Checklist Items
- Completion Standard
- Decision or Output

Optional:

- Evidence Panel
- Gate Decision Panel

## Glossary Page

Required:

- Definition list
- Scope note

Optional:

- Related concepts
- Cross-links
