# Component Library V2

This is the expanded component library for the ETIS Platform Design System.

It supersedes the initial high-level component list by adding component groups, usage rules, and implementation priority.

## Core Rule

Components are engineering communication tools.

They exist to help users understand, decide, produce evidence, and move forward.

## Component Families

1. Orientation
2. Navigation
3. Engineering
4. Readiness
5. Learning
6. Action
