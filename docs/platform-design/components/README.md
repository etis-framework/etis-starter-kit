# ETIS Component Catalog

The ETIS Component Catalog defines the reusable interface and content components used to make ETIS product pages consistent, navigable, and engineering-centered.

This catalog is part of the ETIS Platform Design System. It is not a visual decoration guide. It defines reusable engineering communication patterns.

## Purpose

The Component Catalog exists so every ETIS page can consistently answer:

1. Where am I?
2. Why am I here?
3. What should I do?
4. What comes next?

## Component Groups

| Group | Purpose |
|---|---|
| Orientation Components | Establish where the user is and why the page matters. |
| Navigation Components | Help the user move through the platform. |
| Engineering Components | Communicate engineering work, artifacts, and evidence. |
| Readiness Components | Support gates, decisions, and stage transitions. |
| Learning Components | Help users understand doctrine, concepts, and examples. |
| Action Components | Make next steps clear. |

## Component Rule

A component should exist only when it improves orientation, comprehension, engineering rigor, evidence discovery, or forward progress.
