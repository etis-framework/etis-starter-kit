# Shared Component — Navigation Block

Every Engineering Stage should include clear navigation at page level and stage level.

## Page-Level Navigation

Use this structure near the top of each page:

```markdown
**Engineering Stage:** ES-XXX — Stage Name  
**Current Page:** Page Name  
**Previous:** [Previous Page](previous.md)  
**Next:** [Next Page](next.md)
```

## End-of-Page Navigation

Every page ends with a clear action:

```markdown
## Continue

Continue to [Next Page](next.md).
```

Other acceptable endings include:

```markdown
## Return

Return to [Stage Overview](README.md).
```

```markdown
## Complete

Complete the [Readiness Gate](readiness_gate.md).
```

```markdown
## Next Stage

Continue to [ES-XXX — Next Stage](../ES-XXX/README.md).
```

## Zero Dead-End Rule

No page may simply end.

Every page must tell the engineer what to do next.
