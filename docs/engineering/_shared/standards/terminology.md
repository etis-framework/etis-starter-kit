# Terminology Standard

Consistent terminology is required across the ETIS Engineering Platform.

## Preferred Terms

Use these terms consistently:

| Preferred Term | Avoid |
|---|---|
| Engineering Stage | module, unit, lesson, chapter |
| Engineering Platform | docs, documentation set |
| Artifact | form, worksheet, paper |
| Evidence | proof, note, output, documentation |
| Readiness Gate | quiz, test, checklist |
| Repository-Centered Engineering | repo-first docs, documentation storage |
| Evidence-Centered Engineering | record keeping, compliance paperwork |
| AI Assistance | AI answer, AI solution |
| AI Verification | AI checking, prompt review |
| Stewardship | maintenance only |

## Definitions

Use definitions from `engineering/ES-100/glossary.md` unless a stage requires a local specialization.

## Rule

If a term is shared across multiple stages, define it once and link to the shared glossary.
